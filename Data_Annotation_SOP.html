Image Annotation for Cars and Trees To provide clear instructions for annotating cars and trees in images, while excluding taxis from the car annotations. This SOP applies to the image annotation team responsible for labeling visual data for machine learning purposes. Image annotators must accurately label cars and trees according to the given labels and guidelines. Ensure that your workstation is ergonomically set up to prevent strain from repetitive tasks. 1.0 YYYY-MM-DD Annotator_Name

Instructions and Guidelines

Step_Number Step_Title Instruction Image Example
1 Open Image Annotation Tool Launch the designated image annotation software on your computer. open_annotation_tool.jpg
2 Load Image Import the image to be annotated into the annotation tool workspace. load_image.jpg
3 Identify Cars Carefully examine the image and identify all objects that are cars, excluding taxis. identify_cars.jpg
4 Annotate Cars Use the 'Car' label to draw bounding boxes around each identified car. Ensure the entire car is within the bounding box. annotate_cars.jpg
5 Identify Trees Locate all trees in the image that need to be annotated. identify_trees.jpg
6 Annotate Trees Use the 'Tree' label to draw bounding boxes or polygons around each tree, capturing the full extent of the tree's visible area. annotate_trees.jpg
7 Exclude Taxis Do not annotate vehicles that are clearly marked or recognized as taxis. exclude_taxis.jpg
8 Review Annotations Review all annotations for accuracy and completeness. Make sure no cars or trees have been missed and no taxis have been incorrectly labeled as cars. review_annotations.jpg
9 Save Annotations Save the annotated image in the prescribed format and file naming convention. save_annotations.jpg
10 Submit Annotated Image Upload the annotated image to the designated server or database as per project requirements. submit_annotated_image.jpg
